## 📌 Project Overview

This project analyzes **Delhi's Air Quality Index (AQI) data for the year 2025** using **Python, Pandas, and Power BI**.

The objective is to transform raw environmental monitoring data into meaningful insights through data cleaning, feature engineering, exploratory analysis, and interactive dashboards.

# 📌 Project Workflow

### 1️⃣ Data Collection

### 2️⃣ Data Cleaning

- Removed duplicate records
- Handled missing values
- Converted timestamps
- Standardized column names

- Pollution Level
- Season

- Weekday

- ☀ Average Ozone

- 📅 Monthly AQI Trend
- 🏭 Most Polluted Monitoring Stations
- 🎯 Interactive Filters

---

# 📌 Key Insights

- Delhi experiences the highest AQI during winter.
- Pollution decreases during the monsoon season.
- Several monitoring stations consistently record higher AQI values.
- Hourly trends reveal distinct pollution peaks throughout the day.
